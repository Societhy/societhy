My Project
isarray

Array::isArray for older browsers.

Usage

var isArray = require('isarray');
console.log(isArray([])); // => true
console.log(isArray({})); // => false

Installation

With npm do

$ npm install isarray

Then bundle for the browser with browserify.

With component do

$ component install juliangruber/isarray
