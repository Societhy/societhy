Get the longest item in an array.

Install with npm

npm i longest --save

Install with bower

bower install longest --save

Running tests

Install dev dependencies.

npm i -d && npm test

Usage

var longest = require('longest');
longest(['a', 'abcde', 'abc']);
//=> 'abcde'
longest(['a', 'abcde', 'abc']).length;
//=> 5
